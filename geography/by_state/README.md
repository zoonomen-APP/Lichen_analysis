# Out-of-Bounds Records by State

Each subdirectory contains flagged records for one state, across all herbaria. Within each state, records are classified by error type and sorted by distance from the stated boundary, largest first.

### Error Types

- **COUNTRY** — Coordinates fall outside the United States entirely
- **STATE** — Coordinates fall in a different US state than stated
- **COUNTY** — Coordinates fall outside the stated county but within the correct state

For guidance on interpreting these files, see the [main README](../../README.md).
