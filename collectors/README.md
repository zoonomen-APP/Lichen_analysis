# Out-of-Bounds Records by Collector

Each subdirectory contains flagged records for one collector, organized by state. Records are sorted by distance from the stated county boundary, largest first.

Note that the named collector may not have entered the data. Responsibility for coordinate and county values often lies elsewhere in the data entry chain.

For guidance on interpreting these files, see the [main README](../README.md).
